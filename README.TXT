PROJECT TITLE: Your Recipe Book!
AUTHORS: Jad Dayoub & Timur Turdaliev

PURPOSE OF PROJECT: Organize, add, and remove recipes.
VERSION or DATE: December 21, 2025

HOW TO START THIS PROJECT:
1. Open command prompt.
2. Navigate to the project directory.
3. Make sure project is compiled.
4. Run the program using: java Driver

USER INSTRUCTIONS:
- A menu should be displayed.
- Enter the number of the action you want to perform.
- You can add a recipe a its type, as well as optionally add its ingredients, steps, and tags.
- You can search for an added recipe by title, ingredient, or tag.
- You can choose to rate a recipe out of 5 and choose to scale ingredient amounts based off the number of servings.
- Enter 0 to exit the program.
